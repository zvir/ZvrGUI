trc()function 
public function trc(sender:Object, message:String, key:String, ... args):ZvrTraceMessage

...

Parameters

sender:Object — Object which is reporting.
 
message:String — Message, main subject of report.
 
key:String — Key is used for updating the entry. When for secont time there is trc calles with the same "key", it's updates the message and args of first entry with this key. Main use of this feature is reporting objeets whitch states is changing quckly with a lot of updates e.g loaders
 
... args — additional arguments to be shown in report.

Returns
ZvrTraceMessage — retuns created instance of ZvrTracyMessage, or the one with given "key".

Example 
     trc(this, "Loading", "", "progress, Number(e.bytesLoaded/e.bytesTotal).toFixed(2));
