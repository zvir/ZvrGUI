public function bringToFront(object:DisplayObject, back:uint = 0):int

Brings the DisplayObject to the front of the display list. The back parameter can be used to pull the DisplayObject back a few levels from the front.

Parameters

object:DisplayObject — the DisplayObject to reorder
 
back:uint (default = 0) — the number of levels from the front of the display list

Returns
int — the new index of the DisplayObject
